### Хранение данных GitLab (Omnibus)

В данной сборке GitLab установлен с использованием Omnibus-пакета. Все основные данные, конфигурации и логи хранятся внутри виртуальной машины в стандартных директориях:

#### 1. Основные данные (Repositories, Database, Artifacts)
Путь: `/var/opt/gitlab`

В этой директории хранится самое важное:
- **Репозитории:** `/var/opt/gitlab/git-data/repositories` — здесь хранятся все ваши Git-проекты.
- **База данных:** `/var/opt/gitlab/postgresql/data` — база данных PostgreSQL (пользователи, настройки проектов, комментарии).
- **Артефакты и реестр пакетов:** `/var/opt/gitlab/gitlab-rails/shared` — здесь хранятся CI-артефакты, LFS-файлы и данные Package Registry.
- **Загруженные файлы (Uploads):** `/var/opt/gitlab/gitlab-rails/uploads` — аватары, прикрепленные файлы в Issue/Wiki.

#### 2. Конфигурация
Путь: `/etc/gitlab`

- **`gitlab.rb`**: Главный файл конфигурации. Все изменения настроек должны вноситься именно сюда.
- **`gitlab-secrets.json`**: Содержит ключи шифрования для базы данных. **Критически важный файл для бэкапа.** Без него невозможно будет восстановить зашифрованные данные из БД.
- **`initial_root_password`**: Временный пароль администратора (удаляется через 24 часа после установки).

#### 3. Логи
Путь: `/var/log/gitlab`

Здесь собраны логи всех компонентов GitLab (Nginx, PostgreSQL, Redis, Sidekiq и др.).

---

### Рекомендации по бэкапу

Для создания полной резервной копии GitLab выполните команду внутри виртуальной машины:
```bash
sudo gitlab-backup create
```
Архив будет создан в папке `/var/opt/gitlab/backups`.

**Важно:** Файлы `/etc/gitlab/gitlab.rb` и `/etc/gitlab/gitlab-secrets.json` **не включаются** в стандартный бэкап. Их нужно сохранять отдельно вручную.

### Где хранятся данные на хост-машине (Windows)?

Так как GitLab запущен в виртуальной машине Hyper-V через Vagrant:
1.  **Виртуальный диск:** Все данные хранятся внутри файла виртуального диска (обычно `.vhdx`), который находится в папке, где Hyper-V хранит свои ВМ (или в `.vagrant` подпапке проекта).
2.  **Синхронизированные папки:** Папка проекта на Windows синхронизирована с `/vagrant` внутри ВМ (через SMB). Однако **базы данных и репозитории GitLab НЕ хранятся в этой папке** по соображениям производительности и совместимости файловых систем. Они находятся внутри виртуального диска Ubuntu.
