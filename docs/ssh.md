# Работа с GitLab через SSH

Использование SSH-ключей — это самый безопасный и удобный способ взаимодействия с вашим локальным GitLab. Вам не нужно будет вводить логин и пароль при каждом `git push` или `git pull`.

---

## 1. Генерация SSH-ключа

Если у вас ещё нет SSH-ключа, создайте его:

1. Откройте терминал (PowerShell или Git Bash).
2. Выполните команду:
   ```bash
   ssh-keygen -t ed25519 -C "your_email@example.com"
   ```
   *Если ваша система не поддерживает ed25519, используйте `ssh-keygen -t rsa -b 4096`.*
3. Нажимайте **Enter**, чтобы сохранить ключ в папку по умолчанию (`~/.ssh/id_ed25519`).
4. (Опционально) Введите кодовую фразу (passphrase).

---

## 2. Добавление ключа в GitLab

Чтобы GitLab узнал ваш компьютер:

1. **Скопируйте публичный ключ** в буфер обмена:
   - В Windows (PowerShell):
     ```powershell
     Get-Content $HOME\.ssh\id_ed25519.pub | Set-Clipboard
     ```
   - Или просто откройте файл `id_ed25519.pub` в Блокноте и скопируйте текст.
2. **Зайдите в локальный GitLab** (например, `http://gitlab.local`).
3. Нажмите на свою аватарку (справа вверху) -> **Edit profile**.
4. В левом меню выберите **SSH Keys**.
5. Нажмите **Add new key**.
6. Вставьте ваш ключ в поле **Key**. Поле **Title** заполнится автоматически.
7. Нажмите **Add key**.

---

## 3. Настройка SSH для работы с нестандартным портом (если нужно)

Если вы пробрасываете порты или используете специфические настройки в Vagrant, SSH может работать не по стандартному 22 порту. Однако в данной сборке используется `public_network`, поэтому SSH должен быть доступен по стандартному порту.

Чтобы проверить соединение, выполните:
```bash
ssh -T git@gitlab.local
```
*Если вы видите "Welcome to GitLab, @username!", значит всё настроено верно.*

---

## 4. Клонирование репозитория по SSH

Теперь вы можете клонировать проекты без ввода пароля:

1. Зайдите в проект в GitLab.
2. Нажмите синюю кнопку **Code**.
3. Скопируйте ссылку в разделе **Clone with SSH** (она выглядит как `git@gitlab.local:root/my-project.git`).
4. В терминале выполните:
   ```bash
   git clone git@gitlab.local:root/my-project.git
   ```
